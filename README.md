Учебный 3D Unity-проект: прототип арены для будущей игры **Битва фракций / Battle of Factions**.

---

## О проекте

Это не финальная игра и не сразу MMO.

Сейчас проект развивается как **боевой прототип арены**, на котором постепенно собирается фундамент будущей большой игры.

Главная идея разработки:

> Сначала рабочий результат, потом улучшения.

Проект используется одновременно как:

- учебная практика по Unity и C#;
- основа для собственной игры;
- площадка для проверки боевых и системных механик.

---

## MVP 0.1

Текущий MVP строится вокруг **замкнутого игрового цикла арены**:

**Арена → бой → урон → смерть → лут → волны → победа / поражение**

Цель MVP:

сделать не “всё и сразу”, а рабочую, понятную, расширяемую основу.

---

## Что уже принято по разработке

- сначала делается **MVP арены**;
- проект идёт **по этапам**;
- сначала делается **рабочий фундамент**, потом расширение;
- большие идеи будущей игры не должны ломать текущий этап;
- архитектура постепенно приводится в порядок, без лишнего усложнения;
- New Input System предпочтительнее старого Input Manager;
- решения должны быть простыми, но не одноразовыми.

Как открыть проект
Скачать или клонировать репозиторий.
Открыть папку проекта через Unity Hub.
Выбрать Unity версии:
6000.3.6f1
Дождаться полного импорта.
Открыть сцену арены в:
Assets/_Project/Scenes
Важно

Проект находится в активной разработке.

Возможны:

временные прототипные решения;
не до конца оформленный визуал;
черновые сцены;
служебные папки, которые позже будут почищены.
